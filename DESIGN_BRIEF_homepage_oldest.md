# 言街 — 首页街景地图设计需求文档

## 项目简介

**言街（Kotomachi）** 是一个低压力的日常日语会话练习 Web 应用。用户在一条安静的日本小镇街道上，与三位邻居 NPC 自由聊天，在生活场景中自然习得日语口语。

核心体验关键词：**安静、日常、呼吸感、零压力**。不是游戏，不是课堂，是"去街角便利店买咖啡时顺便说两句日语"的感觉。

## 设计风格体系

### 色彩 Token

| Token | 色值 | 用途 | 情绪 |
|-------|------|------|------|
| background | `#F3EDE0` | 页面底色 | 米白，温暖纸张感 |
| foreground | `#28231A` | 主文字 | 深棕黑，不刺眼 |
| card | `#FAF6EE` | 卡片/气泡底色 | 暖白，柔和 |
| primary | `#2D4A1F` | 主色/用户气泡 | 深绿，沉稳自然 |
| accent | `#C9A84C` | 强调/交互点 | 琥珀，温暖光斑 |
| secondary | `#E8E0CE` | 次色/暖灰 | 柔和过渡 |
| muted | `#D8CFBC` | 弱化背景 | 退后感 |
| muted-fg | `#7A7060` | 弱化文字 | 不争抢 |
| input-bg | `#EDE7D8` | 输入框 | 略深于底色 |
| sidebar | `#1E2A16` | 侧边栏 | 深林绿，安静 |
| sidebar-fg | `#D4C8A8` | 侧边栏文字 | 暖金，不冷 |

### 字体

- **标题**：Noto Serif JP 300 — 衬线体，文雅安静
- **正文**：Noto Sans JP 300/400/500 — 无衬线，清晰可读

### 视觉气质

- 日式和纸质感（米白底 + 微妙纹理）
- 植物系配色（深绿 + 琥珀 + 暖灰），避免高饱和
- 阴影极轻：`0 1px 3px rgba(40,35,26,0.04)` ~ `0 4px 16px rgba(40,35,26,0.08)`
- 圆角统一 `0.75rem`（rounded-xl），不尖锐也不太圆
- 间距宽松，留白充足，呼吸感优先

## 首页功能需求

### 当前首页

目前是一个简单的 CSS 定位地图：米白底色上用 `absolute` 定位放置三个 NPC 卡片，每个卡片显示场所名 + NPC 姓名 + 假名 + ☁️ 状态气泡。视觉过于简陋，缺乏沉浸感。

### 目标首页

**核心目标**：让用户一打开就感受到"走进了一条安静的日本小镇街道"，而不是"看到了三个功能入口"。

#### 必须保留的功能元素

1. **三个 NPC 入口**：点击后进入对应聊天页面
2. **NPC 每日状态气泡**：☁️ + 一句日语独白（同一天不变），像 NPC 此刻的心里话
3. **时段感知**：根据当前时间显示 朝 / 昼 / 夕 / 夜，影响整体氛围
4. **品牌标题**：「言街」+ 副标题

#### 期望的视觉升级方向

- **街景感**：不是俯视地图，而是轻俯视感的街景地图，一条有纵深感的街道。便利店、咖啡馆、居酒屋作为建筑/店面出现在街道两侧
- **时段氛围**：朝（晨光暖黄）→ 昼（明亮柔和）→ 夕（琥珀余晖）→ 夜（深蓝暖灯），背景色调/光影随时间变化
- **微动效**：树叶摇曳、灯光微闪、云朵飘动等轻量 CSS 动画，增加"活着的感觉"
- **交互**：hover 店铺时轻微放大/发光，点击进入聊天

#### 布局参考

- Web 端（主要）：宽屏横版街景，建筑沿街道排列，有透视感
- 移动端：竖版，街道纵向延伸，滚动浏览

#### 禁止事项

- 不要游戏化（不要等级、任务、成就）
- 不要高饱和霓虹色
- 不要卡通 Q 版风格（保持文雅安静）
- 不要弹窗/模态框打断

---

## 技术约束

- **框架**：Next.js 14 + React + Tailwind CSS
- **图片格式**：SVG 优先（矢量场景），PNG 备选（头像等位图）
- **动画**：纯 CSS animation / transition，不引入动画库
- **响应式**：需适配桌面（1280px+）和移动端（375px+）
- **无障碍**：所有交互元素需有 alt / aria-label
